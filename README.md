# Coding Samples

This repository contains selected coding samples adapted from graduate-level coursework in statistics and data science.  
The projects are organized as concise, job-application-ready examples of how I approach applied data analysis, statistical modeling, model diagnostics, and clear communication of results.

Rather than presenting raw homework files, I revised these samples into short technical write-ups that highlight structured analytical workflows: defining the problem, preparing the data, fitting and comparing models, checking assumptions, and interpreting findings.

## Included Samples

### 1. Regression Analysis and Model Diagnostics in R
A structured linear regression workflow using the Duncan occupational prestige dataset, including exploratory analysis, model comparison, and diagnostic checks.

### 2. Logistic Regression and Model Diagnostics in R
A binary classification workflow using generalized linear models, including data preparation, logistic regression, diagnostic checks, and comparison of alternative link functions.

### 3. Flexible Regression and Additive Modeling in R
A comparison of linear, nonparametric, and additive regression approaches to evaluate nonlinear relationships and balance predictive performance with interpretability.

### 4. Censored and Selection Models in R
A comparison of OLS, Heckman selection, and Tobit models for partially observed outcomes, focused on matching the statistical model to the data-generating process.

### 5. Longitudinal Mixed-Effects Modeling in R
A mixed-effects modeling workflow for repeated body-fat measurements, including piecewise time variables, random-effects comparison, and longitudinal interpretation.

## Skills Demonstrated

- Data preparation and exploratory analysis  
- Linear and generalized linear modeling  
- Model comparison and diagnostics  
- Nonlinear and additive regression  
- Censoring and selection models  
- Longitudinal mixed-effects modeling  
- Reproducible analysis in R and Quarto  

## Note

These samples were adapted from coursework for portfolio and job application purposes. They are intended to demonstrate coding style, statistical reasoning, and communication of analytical results in a concise and professional format.
