# Flexible Regression and Additive Modeling in R

This sample compares linear, nonparametric, and additive regression models using U.S. state-level SAT math data.

It includes:
- exploratory analysis of nonlinear trends
- baseline linear regression
- general nonparametric regression
- additive modeling with GAM
- model comparison using RMSE and interpretability

This sample highlights how I evaluate whether added model flexibility is supported by the data.
