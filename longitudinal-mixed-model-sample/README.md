# Longitudinal Mixed-Effects Modeling in R

This sample demonstrates a mixed-effects modeling workflow for repeated body-fat measurements collected before and after menarche.

It includes:
- construction of piecewise time variables
- longitudinal exploratory analysis
- mixed-effects model fitting
- comparison of random-effects structures
- interpretation of subject-specific variation over time

This sample highlights a structured approach to hierarchical and repeated-measures data in R.
