# Censored and Selection Models in R

This sample examines partially observed outcome data using the Long PhD prestige dataset.

It includes:
- a baseline OLS model
- a Heckman selection model
- a Tobit model for left-censored outcomes
- comparison of assumptions across models
- interpretation of which modeling framework best matches the data structure

This sample highlights model choice based on the underlying data-generating process rather than defaulting to standard regression.
